    JAVA 프로그래밍의 기본 단위 : 클래스(메서드,변수,상수를 포함)

    클래스 의사코드(pseudocode) 표현 수학적 표현
    public class 클래스이름 {
      private  자료형변수  변수
      private final  자료형상수  상수
      ...
     
     접근 자료형y 메서드f(자료형x1 x1, 자료형x2 x2, ... , 자료형xn xn) {
            실행 내용
            return y;
        }     		y = f( x1, x2, ..., xn )
    입력 x : 함수에 전달하는 값
    출력 y : 함수의 실행 결과값
    함수 f : x를 바탕으로 실행 후 y를 반환
      ...
    }

    클래스를 이용한 문제 해결 : 트랙 둘레 계산

    ㉯
    		㉮
    ㉱
    		㉰
    		y1 = 트랙둘레계산( 가로길이, 세로길이 )
        = ㉮ + ㉯ + ㉰ + ㉱ = ( ㉮ + ㉱ ) + ( ㉯ + ㉰ )
        = 2×직선부분둘레계산( 가로길이 ) + 원둘레계산( 세로길이 )

    y2 = 직선부분둘레계산( 직선길이 ) = 직선길이

    y3 = 원둘레계산( 지름 ) = PI × 지름

    PI = 3.141592
    public class 둘레계산클래스 { 클래스 시작
        private final 실수 PI = 3.141592; 상수 선언
       
        public 실수 트랙둘레계산( 정수 가로길이, 정수 세로길이 ) { 메서드 시작
            실수 트랙둘레 = 2×직선부분둘레계산( 가로길이 ) + 원둘레계산( 세로길이 ); 실행 내용
            return 트랙둘레;
        }     		메서드 종료
     

        private 정수 직선부분둘레계산( 정수 직선길이 ) {
            return 직선길이;
        }

        private 실수 원둘레계산( 정수 지름 ) {
            실수 원둘레 = PI × 지름;
            return 원둘레;
        }
    } 클래스 끝
